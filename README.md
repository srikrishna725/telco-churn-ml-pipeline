# Telco Customer Churn Prediction

## 📌 Project Overview
This project builds an end-to-end Machine Learning pipeline to predict customer churn using the IBM Telco dataset.

## 🎯 Objective
Predict whether a customer will churn (Yes/No) based on demographic and service usage features.

## 📊 Dataset
IBM Telco Customer Churn Dataset
